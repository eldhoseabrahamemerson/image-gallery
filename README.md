# 🖼️ Image Gallery

A modern and responsive Image Gallery built using **HTML, CSS, and JavaScript**. Users can preview images, navigate using thumbnails or arrow buttons, and enjoy smooth image transitions.

## 🚀 Live Demo

🔗  https://eldhoseabrahamemerson.github.io/image-gallery/

## 📸 Features

- 🖼️ Large image preview
- 📷 Clickable thumbnails
- ⬅️ Previous and Next navigation buttons
- ⌨️ Keyboard navigation using Left and Right Arrow keys
- ✨ Smooth fade transition between images
- 🎯 Active thumbnail highlight
- 📱 Responsive design for desktop and mobile
- 🎨 Clean and modern user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JavaScript)

## 🎮 How to Use

1. Open the website.
2. Click any thumbnail to display it as the main image.
3. Use the **Previous** and **Next** buttons to browse through the gallery.
4. You can also use the **Left** and **Right Arrow** keys on your keyboard to navigate between images.
